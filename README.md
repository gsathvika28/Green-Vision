A prototype AI system to detect black gram leaf diseases from images and suggest remedies. Built with **PyTorch** and **Streamlit**.

## Features

- Upload single or multiple leaf images.
- Predict disease among 5 classes:
  - Anthracnose
  - Healthy
  - Leaf Crinkle
  - Powdery Mildew
  - Yellow Mosaic
- Display confidence scores with animated bars.
- Suggest remedies for each disease.
- Download CSV report for multiple images.
- Premium light-themed UI with green accents.
